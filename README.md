# 🤖 Agentic AI Project

A multi-agent AI system built with **Microsoft AutoGen** and **Model Context Protocol (MCP)**.
Agents can chat, solve problems, and interact with the filesystem — all powered by **free APIs**!

---

## 🚀 Features

- 🧠 Multi-agent conversation using AutoGen AgentChat
- 📁 MCP Filesystem integration — agents can create and read files
- 🔄 RoundRobin Group Chat between multiple agents
- 🆓 Powered by free Groq & Gemini APIs (no OpenAI billing needed!)
- 💬 Interactive Student-Tutor agent setup

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Core language |
| AutoGen AgentChat | Multi-agent framework |
| MCP (Model Context Protocol) | Filesystem tool integration |
| Groq API | Free LLM (llama-3.3-70b) |
| Gemini API | Free LLM with reliable tool calling |
